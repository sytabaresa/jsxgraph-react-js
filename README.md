# jsxgraph-react-js

## Install

```bash
npm install --save jsxgraph-react-js
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'jsxgraph-react-js'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
